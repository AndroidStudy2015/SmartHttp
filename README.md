# SmartHttp
基于retrofit2.0 支持下载、上传、post、get的小框架

```
-------------------服务器地址切换-------------------
 服务器地址类型：1-本地 2-公测 3-生产
serverUrlType=3
# 本地环境url-1
LocalBeatUrl="http://www.xxx.com";
# 公测环境url-2
OpenBeatUrl="http://www.xxx.com";
# 生产环境url-3
ProductionUrl="http://www.xxx.com";
# 控制log打印
isPrintLog=true
```
